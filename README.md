## Hotel Booking App

Hotel Booking app made with Flutter and Firebase

## Development Setup

Register in Razor Pay and get a test key,

Add the key at 

```
/lib/controllers/confirmation_controller.dart/line 153
```

Clone the repository and run the following commands:
```
flutter pub get
flutter run
```
